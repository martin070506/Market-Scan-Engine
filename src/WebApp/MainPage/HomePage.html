What this tool does

This interface is your gateway to a Python-powered engine that reads your CSV data, validates it, and searches for structural patterns in price.

  • ①

    CSV-first workflow

    Export OHLCV data from your broker or platform as a CSV file, then drag and drop it here. The backend will parse and clean your data automatically.

  • ②

    Pattern-oriented analysis

    The engine focuses on structural setups like cup-and-handle: uptrends, controlled pullbacks, rounded bases, and breakout zones.

  • ③

    Actionable summaries

    Instead of just numbers, you’ll get a summary of candidate patterns, quality scores, and timing hints – ready to plug into your workflow.

Example: Cup & Handle Scan
  • AAPL Strong base Depth: 24% · Duration: 6 weeks
  • TSLA On watch Handle forming · Tight ranges
  • NVDA Skip Too volatile · V-shaped

How it works

Behind this interface is a Python stack that processes your CSV file step by step – from validation and parsing to pattern detection and scoring.

1

Upload your CSV

Drag and drop a CSV containing OHLCV data (timestamp, open, high, low, close, volume). The interface sends it securely to the backend via an HTTP request.

2

Python analysis pipeline

The server (Python) reads your file with pandas, normalizes timestamps, checks data quality, then runs pattern-detection logic on your time series.

3

Pattern summary returned

Once the engine completes, it returns structured JSON – which the UI can turn into tables, charts, or alerts about potential setups.
